# DesignKP 2D (Viewport پایه)

این نمونه دقیقاً همون چیزی رو که گفتی پیاده می‌کنه:
- Grid شطرنجی (ریز/درشت) + محورهای X/Y + مبدا 0,0
- زوم با اسکرول (حول نقطه‌ی موس)
- پن با نگه‌داشتن و درگِ کلیکِ چرخ ماوس (middle mouse)
- تعیین اینکه «هر ۱ متر چند مربع» باشد
- تغییر رنگ گرید/محورها
- انتخاب واحد نمایش mm / cm (واحد داخلی همیشه mm)

## ساختار فایل‌ها
- `index.html`
- `main.js`
- `src/engine/*`

نکته: در وضعیت فعلی، استایل‌ها عمدتاً داخل خود `index.html` هستند و `styles.css` عملاً استفاده نمی‌شود (اگر خواستی می‌تونیم تمیزش کنیم یا به HTML لینک کنیم).

## اجرا روی ویندوز (مسیر پروژه)
تو گفتی مسیر پروژه:

`C:\DesignKP\2d`

پس کل پوشه‌ی این دمو رو کپی کن داخلش (یا همین فایل‌ها رو اونجا بساز).

## اجرای درست
به خاطر اینکه `main.js` با `type="module"` لود میشه، باید با یک وب‌سرور محلی اجرا بشه (باز کردن مستقیم فایل ممکنه به خاطر CORS/Modules مشکل بده).

### روش 1: Python
داخل پوشه اجرا کن:

```bat
cd C:\DesignKP\2d
python -m http.server 5173
```
بعد در مرورگر:

`http://localhost:5173`

### روش 2: VS Code Live Server
روی `index.html` بزن Live Server.

## نکته‌ی مهم معماری
- **World unit = mm** (برای دقت کابینت و CNC بهتره)
- UI فقط تبدیل نمایش انجام میده (cm/mm)
- دوربین با `(xMm, yMm, zoom)` کنترل میشه

مرحله‌ی بعدی که منطقیه: اضافه کردن ابزار Wall (با snap به grid و endpoint).
